# GithubJapanese
GitHubを日本語化するユーザスクリプトです。
## 注意
+ この日本語化はHtml上のテキストを置換しているものであり、公式の翻訳及び提供ではありません。
+ この日本語化は不完全なものであり、今後もアップデートされる予定です。
+ GitHub公式のレイアウトの変更により、この翻訳が崩れる可能性があり、テキストが誤植される可能性がございます。
## 概要
GitHubの非公式日本語翻訳です。
